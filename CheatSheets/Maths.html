CBSE Class 10 Maths – Formula & Concept Sheet

(For last-minute revision before the exam)

1. Real Numbers

Euclid’s Division Lemma: a = bq + r, 0 ≤ r < b
  • HCF = last non-zero remainder
  • p/q terminates ⇨ q = 2m × 5n
  • Otherwise: non-terminating recurring decimal

2. Polynomials

For ax² + bx + c:
Sum of zeroes = −b / a
Product of zeroes = c / a

3. Pair of Linear Equations

a₁x + b₁y + c₁ = 0
a₂x + b₂y + c₂ = 0
  • Unique solution: a₁/a₂ ≠ b₁/b₂
  • Infinite solutions: a₁/a₂ = b₁/b₂ = c₁/c₂
  • No solution: a₁/a₂ = b₁/b₂ ≠ c₁/c₂

4. Quadratic Equations

ax² + bx + c = 0
x = [ −b ± √(b² − 4ac) ] / 2a
  • D > 0 → two real roots
  • D = 0 → equal roots
  • D < 0 → no real roots

5. Arithmetic Progressions

nth term: aₙ = a + (n − 1)d
Sum: Sₙ = n/2 [ 2a + (n − 1)d ]

6. Triangles

  • Similarity criteria: AAA, SAS, SSS
Pythagoras Theorem: Hypotenuse² = Base² + Perpendicular²
Area ratio = (Side₁ / Side₂)²

7. Coordinate Geometry

Distance = √[(x₂ − x₁)² + (y₂ − y₁)²]
Section Formula (internal):
x = (mx₂ + nx₁)/(m + n)
y = (my₂ + ny₁)/(m + n)
Area of triangle = 1/2 | x₁(y₂ − y₃) + x₂(y₃ − y₁) + x₃(y₁ − y₂) |

10. Circles

  • Tangent is perpendicular to radius at point of contact
  • Number of tangents:
    • Inside → 0
    • On → 1
    • Outside → 2

12. Areas Related to Circles

Circumference = 2πr
Area = πr²
Sector area = (θ / 360) × πr²
Arc length = (θ / 360) × 2πr

13. Surface Areas and Volumes

Cube: TSA = 6a², Volume = a³
Cuboid: TSA = 2(lb + bh + hl), Volume = lbh
Cylinder: TSA = 2πr(r + h), Volume = πr²h
Cone: TSA = πr(l + r), Volume = (1/3)πr²h
Sphere: Surface Area = 4πr², Volume = (4/3)πr³

14. Statistics

Mean = Σfᵢxᵢ / Σfᵢ
Median = l + [ (N/2 − cf) / f ] × h
Mode = l + [ (f₁ − f₀) / (2f₁ − f₀ − f₂) ] × h
Empirical Formula: Mode = 3 × Median − 2 × Mean

15. Probability

P(E) = Number of favourable outcomes / Total outcomes
0 ≤ P(E) ≤ 1
P(not E) = 1 − P(E)
